# 🌊 Water Quality Prediction System

Hệ thống dự đoán chất lượng nước sử dụng Machine Learning và Streamlit.

## 📋 Mô tả
Ứng dụng này giúp dự đoán chất lượng nước dựa trên các thông số như:
- pH
- Nhiệt độ (Temperature)
- Oxy hòa tan (Dissolved Oxygen)
- Độ đục (Turbidity)

## 🚀 Cài đặt

### Yêu cầu
- Python 3.8+
- Git

### Các bước cài đặt
1. Clone repository:
```bash
git clone <your-repo-url>
cd Water_Quality
```

2. Cài đặt các thư viện:
```bash
pip install -r requirements.txt
```

3. Chạy ứng dụng:
```bash
streamlit run app.py
```

4. Mở trình duyệt tại: `http://localhost:8501`

## 📁 Cấu trúc dự án
